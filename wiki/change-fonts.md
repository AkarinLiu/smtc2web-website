# 更改字体
> [!CAUTION] 
> **为什么要这样做？**
>
> 因为之前的版本使用的字体默认使用系统内置的微软雅黑，这会导致你无形中被暴露在字体侵权风险之下。
>
> **如果你用的是 v0.2.1 之前的版本，请按照以下方式更改字体：**
## OBS Studio

在 “源” 中双击 “浏览器源”，在自定义 CSS 中添加元素，例如：

```
body { background-color: rgba(0, 0, 0, 0); margin: 0px auto; overflow: hidden; font-family: MiSans;}
```

> [!TIP] 注意
> 此更改只对已安装至系统的字体有效，如需要查找已安装的字体，请按照以下路径访问：开始 ➡️ 设置 ➡️ 个性化 ➡️ 字体。
